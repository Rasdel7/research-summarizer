# Research Paper Summarizer 📚

AI-powered academic tool that summarizes
research papers into structured insights
using Google Gemini.

## Live Demo
[Click here](YOUR_STREAMLIT_URL)

## Features
- Paste text, upload PDF or use sample papers
- Structured summary: problem, methodology,
  findings, contributions, limitations
- 4 summary types and 3 depth levels
- Q&A generation for comprehension testing
- Side-by-side paper comparison
- Paper history with reading stats
- Download summaries as .txt
- Built-in reading guide and must-read list
- 3 sample papers: Transformer, BERT, ResNet

## Tools Used
- Python, Streamlit, Google Gemini API,
  PyPDF2, Plotly, Pandas

## How to Run Locally
pip install streamlit google-generativeai PyPDF2 pandas plotly
streamlit run app.py
